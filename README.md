# Raul Pizza — Pizzeria Website

A responsive landing page for a fictional pizzeria in Craiova, built from scratch with a functional shopping cart. My most complete project — the first one with real state management in JavaScript.

## Built with
- HTML5
- CSS3 (Grid, Flexbox, custom properties, scroll-snap)
- JavaScript (DOM manipulation, array methods, state management)
- Bootstrap Icons

## Features
- Fully responsive design (mobile, tablet, desktop)
- Mobile hamburger navigation
- **Functional shopping cart:**
  - Add pizzas to cart from the menu
  - Automatic quantity grouping (same item shows as x2)
  - Live total calculation
  - Remove items from the cart
  - Slide-in cart panel
- Reviews carousel with swipe (mobile) and arrow navigation (desktop)
- Menu section with 8 pizzas
- Footer with schedule, contact info, and social links

## What I learned
This project was my first time managing changing state in JavaScript — keeping a data list as the single source of truth and rendering the UI from it. The cart logic (find / filter / forEach) is the same thinking used in React.
